Dog API

Description: This project is a web application that displays a random dog image each time the page is loaded. It uses the Dog CEO public API to fetch dog images. The main objective of this project is to demonstrate how to make HTTP requests using fetch and how to dynamically manipulate the DOM with JavaScript.
